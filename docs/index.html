AnkiLingual

Anki is a powerful tool for learning languages. AnkiLingual makes it easy to add new vocabulary cards with images to your Anki decks from any website.

Install the extension for Chrome, Firefox, or Edge.

Getting Set Up

You will need the Anki desktop app installed on your computer to use, and the AnkiConnect add-on. As long as you have Anki installed, the extension will prompt you to install AnkiConnect.

Anki must be running when you use the extension. If you run into issues, add an issue on Github.

Adding Images

Simply right-click on an image and select "Add to Card" to add the image to your in-progress card.
